# Отношение потребителей к использованию искусственного интеллекта в дерматологии
Данный репозиторий содержит скрипты, написанные и использованные в рамках курсовой работы "Отношение потребителей к использованию искусственного интеллекта в дерматологии", выполненной студентами 2 курса НИУ ВШЭ ОП "Экономика и анализ данных" Дугаевым Дмитрием, Космыниным Александром, Малакшанидзе Анной.

Далее будет описана последовательность действий для самостоятельной проверки гипотез и получения результата.

## Очистка данных
1. Скачайте результаты опроса, их можно найти в файле `Отношение к использованию искусственного интеллекта в дерматологии.csv`
2. Загрузите в R вышеупомянутый датафрейм, примените скрипт `pipeline.R`. Вы получите очищенный файл `cleaned_data_dermatology.xlsx`
3. Откройте файл `PipelineExcel.md`, проследуйте по инструкции по работе с MS Excel PowerQuery.
4. Откройте скрипт `reshape_data.R`, исполните его. Вы получите готовый для построения модели датасет.
