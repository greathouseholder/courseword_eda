# Pipeline обработки очищенной таблицы с данными в Excel
Данный pipeline поможет получить данные в wide-формате

## Последовательность действий
1. Оставим только следующие столбцы: 
    - столбец с ID
    - столбец с чётностью минуты
    - `"Обращались ли Вы к врачу-дерматологу?"`
    - `q6` (это столбец `"Я доверяю врачу больше, чем алгоритмам ИИ."`)
    - столбцы выбора в DCE
2. Переименуем столбцы:
    - столбец с чётностью минуты — `minute`
    - `"Обращались ли Вы к врачу-дерматологу?"` — `experience`
    - `q6` — `trust`
    - столбцы выбора в DCE: 31, 32, ..., 46
3. Поставим курсор на А1, откроем вкладку "Данные", "Из таблицы", "Ок", откроем PowerQuery
4. Добавим чётность минусы (пользовательский столбец `is_minute_even`)

```
= if [minute] = "Четная" then true else false
```

5. Удалим столбец `minute`
6. Добавим пользовательский столбец `Selections`

```
if [is_minute_even] = true
then {[31], [32], [33], [34], [35], [36], [37], [38]}
else {[39], [40], [41], [42], [43], [44], [45], [46]}
```

7. Удалим столбцы выбора в DCE
8. Развернём столбец `Selections`
9. Добавим столбец `Индекс` (от 1)
10. добавляем столбец `trial` со следующей формулой

```
= Number.Mod([Индекс]-1, 8) + 1
```

11. Удалим столбец `Индекс`
12. Добавим столбец `choice` со следующей формулой

```
if [Selections] = "ИИ" then 1
else if [Selections] = "Доктор" then 2
else 3
```

13. Добавим столбцы `provider1`, `provider2`, `provider3` (ai, doctor, doctor&ai)
14. Добавим столбец `price` в зависимости от цены

```
if [is_minute_even] = true then 1500 else 3500
```

16. Удалим столбцы `is_minute_even` и `Selections`
17. Убедимся, что столбцы идут в следующем порядке: `ID`, `experience`, `trust`, `trial`, `choice`, `provider1`, `provider2`, `provider3`, `price`
17. "Главная" -> "Закрыть и загрузить в..." -> "Загрузить"
18. В самом Excel добавим столбцы `accuracy1`, `accuracy2`, `accuracy3`, `time1`, `time2`, `time3`
19. Циклично вставим туда данные из файла `cycled_data.xlsx`
20. Сохраним новый файл с названием `cleaned_data_wide.xlsx`

**Готово!**
