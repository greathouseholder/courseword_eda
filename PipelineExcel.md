# Pipeline обработки очищенной таблицы с данными в Excel
Данный pipeline поможет получить данные в wide-формате

## Последовательность действий
1. Удаляем все столбцы, кроме столбца с чётностью минуты, столбцов выбора в DCE, а также оставляем столбцы "Обращались ли Вы к врачу-дерматологу?" и "Я доверяю врачу больше, чем алгоритмам ИИ." (он же q6)
2. Оставшиеся переименовываем, чтобы названия были короткие (так удобнее)
3. Ставим курсор на А1, данные, из таблицы, открываем PowerQuery
4. добавляем столбцей индексов (с 1)
5. добавляем чётность минусы (is_minute_even с помощью формулы (пусть название столбца минуты — minute)

= if [minute] = "Четная" then true else false

6. удаляем столбец minute
7. добавляем столбец Selections с формулой (буквы - названия столбцов выбора DCE)

if [is_minute_even] = true
then {[31], [32], [33], [34], [35], [36], [37], [38]}
else {[39], [40], [41], [42], [43], [44], [45], [46]}

8. удаляем столбцы выбора в DCE
9. разворачиваем столбец Selections
10. добавляем столбец Индекс (от 1)
11. добавляем столбец trial со следующей формулой

= Number.Mod([Индекс]-1, 8) + 1

12. удаляем столбец Индекс
13. добавляем столбец choice со следующей формулой

if [Selections] = "ИИ" then 1
else if [Selections] = "Доктор" then 2
else 3

14. добавляем столбцы provider1, provider2, provider3
15. добавляем price в зависимости от цены

if [is_minute_even] = true then 1500 else 3500

16. можем удалить столбцы is_minute_even и Selections
17. главная -> закрыть и загрузить в... -> загружаем и сохраняем
